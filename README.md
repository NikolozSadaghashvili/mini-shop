# Mini-Shop

Mini-Shop is a simple full-stack e-commerce application built with React (frontend) and Node.js + Express + MongoDB (backend). It supports user authentication, product management, and a shopping cart.

## Features

- User registration and login with JWT authentication
- Add, view, and manage products
- Shopping cart with quantity management and local storage persistence
- Profile page showing user info and their added products
- Responsive design with CSS
- Toast notifications for actions

## Technologies Used

- **Frontend:** React, React Router, Axios, React Toastify
- **Backend:** Node.js, Express.js, MongoDB, Mongoose, JWT, Bcrypt
- **Styling:** CSS
- **Version Control:** Git
